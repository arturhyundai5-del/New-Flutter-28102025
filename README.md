Задание на сегодня: 
1. Добавьте в свой проект пакет SharedPreferences.
Для этого в терминале пропишите команду: flutter pub add shared_preferences . Эта команда добавит пакет в зависимости проекта.
После вам нужно загрузить все пакеты через команду: flutter pub get .
Теперь вы можете импортировать пакет в любой файл проекта, через import 'package:shared_preferences/shared_preferences.dart';

2. Используйте SharedPreferences в вашей программе (запись и чтение).
Примеры использования можете найти на сайте: https://pub.dev/packages/shared_preferences
